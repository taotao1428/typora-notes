## 常用命令

## grep



```
-n 显示行号
-<> 输出匹配行的上下行 -2 打印上下2行
-c 只显示匹配的次数
```



如果使用正则表达式，可以使用egrep，模式中不需要反斜杠







## awk

每一行表示一条记录



awk默认的分割符是透明字符

```
-F':' 修改分割符，默认为透明字符 awk -F '[ ,]+' 可以使用正则表达式
```



```
NR表示当前行号
NF表示当前行的字段数

```



![img](/linux/.assert/常见命令/1089507-20170126225731300-1873454323.jpg)



使用begin，end可以实现跟丰富的功能

```
awk 'BEGIN{print "hello"} {print $0} END{print "world"}' test.txt
```
